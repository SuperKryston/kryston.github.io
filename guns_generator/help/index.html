Portable

The application has a portable feature as well, to disable this feature, delete the file named portable, all files will then be stored at C:\Users\<Your Username>\AppData\Roaming\Kryston\Guns Generator

 

Debug window

The debug window essentially shows how variables are handled in the generator, there are also useless features but we don’t talk about those ;)

To access the debug window, a text file needs to be created as the example shows

In the text file, options can be specified for example text color, background color and window name, it formats as the following

[debug]

text=lime

background=black

name=Debug screen - Guns generator v1.0
